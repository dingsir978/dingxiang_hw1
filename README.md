创建新矩阵，遍历每个元素进行相加

# 3.矩阵乘法 mul_matrix

**​思路**​：

检查a的列数是否等于b的行数
三重循环计算点积：result[i][j] = Σ(a[i][k] * b[k][j])
### 实例
![矩阵乘法](images/矩阵乘法.png)
